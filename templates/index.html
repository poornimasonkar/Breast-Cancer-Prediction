🎀 Breast Cancer Prediction Model 🎀

ℹ️ About This App

  • 🤖 Built using Machine Learning (Logistic Regression Model)
  • 🧪 Predicts whether a tumor is Cancerous or Non-Cancerous
  • 🧬 Takes 30 numerical features related to tumor characteristics
  • 💻 Developed with Python, Flask, and HTML
  • 📂 Uses a trained model saved via Pickle for fast prediction
  • 👩‍⚕️ Supports early detection of breast cancer
  • ✅ User-friendly input and clear prediction results

📄 Model Summary

  • 📊 Algorithm Used: Logistic Regression
  • 🧠 Why Logistic Regression? Simple yet powerful model used for binary classification tasks like predicting cancer vs. non-cancer.
  • 📈 Training Accuracy: 96.5%
  • 📂 Dataset Source: UCI Breast Cancer Wisconsin Diagnostic Dataset
  • 🎯 Performance Metrics:
    • ✔️ Accuracy: 96.5%
    • ✔️ Precision: 95%
    • ✔️ Recall: 94%
    • ✔️ F1-Score: 94.5%
📚 Did You Know? Early detection of breast cancer significantly improves the chances of survival. This app is not a substitute for professional diagnosis, but it supports awareness. 💡
